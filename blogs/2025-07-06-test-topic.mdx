### Getting Started: Your First Automated Payout

Let's dive into the code. Automating a payment is incredibly simple. All you need is your API key, which you can generate from your [FlashUSDT Dashboard](https://flashusdtsender.xyz/dashboard).

Here’s how to send a single 100 USDT payment to a TRC20 address using Python.

<CodeBlock language="python" title="Example: Sending a Single USDT Payment">
{`
import requests
import json

# Your FlashUSDT API credentials
API_KEY = "YOUR_API_KEY"
API_SECRET = "YOUR_API_SECRET"

# API Endpoint
url = "https://flashusdtsender.xyz/api/v1/send"

# Payment details
payload = {
  "network": "TRC20",
  "recipients": [
    {
      "address": "TX...YourRecipientAddress...XYZ",
      "amount": 100.00
    }
  ]
}

headers = {
  "Content-Type": "application/json",
  "X-API-KEY": API_KEY,
  "X-API-SECRET": API_SECRET
}

try:
  response = requests.post(url, headers=headers, data=json.dumps(payload))
  response.raise_for_status()  # Raise an exception for bad status codes

  result = response.json()
  print("Successfully initiated payment!")
  print(f"Transaction Batch ID: {result.get('batch_id')}")
  print(f"Status: {result.get('status')}")

except requests.exceptions.HTTPError as err:
  print(f"HTTP Error: {err}")
  print(f"Response Body: {err.response.text}")
except Exception as e:
  print(f"An error occurred: {e}")

`}

### Scaling Up: Batch Payouts for Mass Payments

The real power of FlashUSDT shines when you need to pay multiple people at once. Our API is optimized for bulk sending, allowing you to process hundreds of payments in a single request.

Imagine you have a list of affiliate payouts in a simple CSV file (`payouts.csv`):
```csv
address,amount,network
TV...Address1,55.50,TRC20
0x...Address2,120.00,BEP20
TY...Address3,75.25,TRC20
```

You can write a simple script to read this file and send all payments in one go.

<CodeBlock language="python" title="Example: Processing Batch Payouts from a CSV">
{`
import requests
import json
import csv

# Your FlashUSDT API credentials
API_KEY = "YOUR_API_KEY"
API_SECRET = "YOUR_API_SECRET"

# API Endpoint
url = "https://flashusdtsender.xyz/api/v1/send"

# Read recipients from CSV
recipients = []
with open('payouts.csv', mode='r') as csv_file:
    csv_reader = csv.DictReader(csv_file)
    for row in csv_reader:
        recipients.append({
            "address": row["address"],
            "amount": float(row["amount"]),
            "network": row["network"]
        })

# Create the payload with multiple recipients across different networks
payload = {"recipients": recipients}

headers = {
  "Content-Type": "application/json",
  "X-API-KEY": API_KEY,
  "X-API-SECRET": API_SECRET
}

# Send the batch transaction
try:
    response = requests.post(url, headers=headers, data=json.dumps(payload))
    response.raise_for_status()
    result = response.json()
    print("Successfully initiated batch payout!")
    print(f"Transaction Batch ID: {result.get('batch_id')}")
    print(f"Check status at: https://flashusdtsender.xyz/dashboard/history/{result.get('batch_id')}")

except requests.exceptions.HTTPError as err:
    print(f"HTTP Error processing batch: {err}")
    print(f"Response Body: {err.response.text}")

`}
</CodeBlock>

This script reads your CSV, constructs a single API request, and sends all payments simultaneously. This is the power of automation—turning hours of manual work into seconds of script execution.
